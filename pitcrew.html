Patch File
  • Release containing the patches with workface corner or block walls for orientation

Engraving / Scribe File
  • Scribe lines file- Final Block file with one colour scribe line with greyed out block


  • Part Engraving file- Final Block file with part engraving

Gundrill File
  • Finish file just labelled for gundrill work

Patch File - Orange
  • Covers to Aid Machining
  • Pin Covers, Pocket Covers, Hole Covers
  • Machining can remove to cut under those areas if that area is approved for machining
  • Machining can change to Green patches if area will require EDM

Pin Pads - Light Blue
  • 3D Covers for Pin Reams
  • 0.125” Offset in height and diameter
  • Provides a flat surface stock for drilling
  • Machining can remove once completed


EDM Patch - Green
  • Covers that have been identified by Engineering/Pit Crew that will should be finished by EDM
  • Machining verifies areas will be EDM
  • Machining can change to Orange if it can be machined
  • Green Covers are provided to EDM department

Do Not Remove Patch - White
  • Covers identified by Engineering/Pit Crew that cannot be finished at this time
  • Engineering/Pit Crew will change colour once machining approval happens to the area
